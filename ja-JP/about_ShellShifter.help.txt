    各種シェルを起動する為の機能を提供します

SHORT DESCRIPTION
    各種シェルを起動する為の機能を提供します

LONG DESCRIPTION
    特別な設定をしなくても自動的にインストール済みのアプリケーションを判別し、各種シェルを起動することが出来ます。
    cmd と powershell には32ビット版と64ビット版がありますがそれらをコマンドラインから明示的に呼び出すこともできます。

EXAMPLES
    cmd32 /c 'dir'
    cmd64 /c 'dir'
    posh32 -c 'dir'
    posh64 -c 'dir'
    msys -c 'ls'
    mingw32 -c 'ls'
    mingw64 -c 'ls'
    cygwin -c 'ls'
    gitbash -c 'ls'
